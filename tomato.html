RottenTomatoes API

21 MAR 2015

This one of my favorite projects as I find myself constantly wanting to watch a new movie but too lazy to look for a movie with good reviews. So instead I wrote a program that filtered the good movies for me!

Ths project has four main features:

    1. Movie Search
    2. Display top 10 most popular current rentals.
    3. Display movies with a minimum review score and minimum amount of reviews.
    4. Display movies with a minumimim review score. (takes the average of the critic and audience score)


1.) Movie Search

Pretty self explantary, build the url, parse the JSON result, print desired data.

def movie_search() -> None:
    """
    1.) Search movie: returns movie title, critic score, audience score
    """
    while True:
        movie = input('Movie: ')
        url = movie_api.build_url(movie,3,movie_search_url)
        json_result = movie_api.json_response(url)
        print()
        if json_result['total'] != 0:
            for movie in json_result['movies']:

                print('{} ({})'.format(movie['title'],movie['year']))
                print('Critics Rating: {}'.format(_ratings(movie['ratings']['critics_score'])))
                print('Audience Rating: {}%'.format(movie['ratings']['audience_score']))
                print()
            break

        print('Sorry, no results for "{}", try again'.format(movie))
        print()
